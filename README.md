# agent-skills

A collection of skills for GitHub Copilot CLI and other AI coding agents.

## Skills

### zoxide

Smart directory navigation using [zoxide](https://github.com/ajeetdsouza/zoxide) inside Copilot CLI. Jump to frequently-used directories without typing full paths.

**Install:**

```bash
npx skills add TransientError/agent-skills
```

**Prerequisites:** [zoxide](https://github.com/ajeetdsouza/zoxide) must be installed on your system.

**Usage:** Just say `z <query>` in your Copilot CLI session.
